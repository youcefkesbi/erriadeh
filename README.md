# الريادة (Erriadeh)

منصة تواصل خريجي المدرسة — Vue 3 + Supabase.

## المتطلبات

- Node.js 18+
- مشروع Supabase مع الجداول والتخزين المُعدّ مسبقاً

## الإعداد

1. استنساخ المشروع وتثبيت التبعيات:

```bash
npm install
```

2. إنشاء ملف `.env` في الجذر (انسخ من `.env.example`):

```
VITE_SUPABASE_URL=رابط_مشروع_سوبابيس
VITE_SUPABASE_ANON_KEY=مفتاح_الأنون_الخاص_بك
```

3. في Supabase:
   - إنشاء bucket باسم **Transcript** في Storage (عام) والسماح بالرفع للمصادقين إن لزم.
   - تفعيل RLS على `profiles`. السياسات المطلوبة لتسجيل الطلاب وحفظ رابط كشف الدرجات:
     - **INSERT**: `auth.uid() = id` (إنشاء الملف الشخصي بعد التسجيل).
     - **UPDATE**: `auth.uid() = id` (تحديث الحساب، بما فيه `transcript_url` بعد رفع الملف).
     - **SELECT**: المستخدم يقرأ صفّه فقط؛ المدير يقرأ الكل حسب سياساتك.
   - تعيين `role = 'admin'` لأحد الحسابات في `profiles` لاستخدام لوحة الإدارة.

4. تشغيل المشروع:

```bash
npm run dev
```

## الصفحات

- **الرئيسية** `/` — روابط تسجيل الدخول وإنشاء حساب
- **تسجيل الدخول** `/login`
- **إنشاء حساب** `/signup` — الاسم، سنة التخرج، الهاتف، رفع صورة كشف الدرجات
- **لوحة الطالب** `/dashboard` — للطلاب النشطين فقط (الملف الشخصي + الإعلانات)
- **لوحة الإدارة** `/admin` — للمدير: قائمة المعلّقين، موافقة/رفض

## البناء

```bash
npm run build
```

الملفات الجاهزة للنشر في `dist/`.
